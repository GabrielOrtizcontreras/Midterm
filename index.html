Follow the prompts on the boxes in the example above. Add image, header text, nav and footer info. Please recall that all text still has to be marked up with h1, h2, p, etc.

Add a third hyperlink to a website of your choice. The nav is an unordered list, but the links display without any bullets or underlining. Add a style for the hover behavior of the anchor or a tag.
